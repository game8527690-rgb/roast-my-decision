# 🔥 Roast My Decision

**Roast My Decision** is a fun, AI-powered web application that takes your choices—big or small—and gives you a brutally honest, sarcastic, and hilarious "roast" about them. 

Built with modern web technologies, this project combines a sleek UI with AI intelligence to provide users with a unique (and sometimes painful) perspective on their decision-making.

---

## 🚀 Features

- **Decision Roasting:** Enter any decision and get an instant AI-generated roast.
- **Modern UI/UX:** Clean, responsive design built with Tailwind CSS.
- **AI Integration:** Leveraging **Grok API** (xAI) to generate witty and context-aware responses.
- **Fast Performance:** Built on Next.js for a seamless user experience.

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **AI Engine:** Grok API (xAI)
- **Deployment:** [Vercel](https://vercel.com/)

## 🏁 Getting Started

To run this project locally, follow these steps:

### 1. Clone the repository
`git clone https://github.com/game8527690-rgb/roast-my-decision.git`
`cd roast-my-decision`

### 2. Install dependencies
`npm install`

### 3. Set up environment variables
Create a **.env.local** file in the root directory and add your API keys:
`XAI_API_KEY=your_xai_key_here`

### 4. Run the development server
`npm run dev`

Open http://localhost:3000 with your browser to see the result.
